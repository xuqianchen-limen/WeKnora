# 知识助理 - Chrome 浏览器扩展插件需求文档

## 1. 产品概述

**产品名称**：知识助理
**产品形态**：Chrome 浏览器扩展插件（Manifest V3）
**核心定位**：知识库问答 + 网页内容快速采集 + Markdown 速记

## 2. 设计规范

| 项目 | 说明 |
|------|------|
| 设计风格 | 扁平化 |
| 主体色 | `#07C160` |
| 图标 | 绿色背景 + 白色线条书本图标 |
| 字体 | 系统默认（PingFang SC / Microsoft YaHei） |

## 3. 功能模块

### 3.1 双账号登录

插件安装后首页展示两种账号登录入口：

#### 3.1.1 知识助理账号

- 使用微信头像账号体系
- 点击后**自动登录**（读取微信授权信息，无需手动操作）
- 登录过程展示头像和账号信息的加载过渡

#### 3.1.2 WeKnora 自建服务账号

- 需要手动填写**服务地址**和 **API Key**
- 提供「测试连接」功能，验证配置是否正确
- 填写完成并验证通过后点击「保存并登录」进入主界面

### 3.2 登录后主界面

登录成功后进入主界面，顶部显示：

| 元素 | 说明 |
|------|------|
| 头像 | 知识助理账号显示微信头像；WeKnora 账号显示字母头像（蓝底白字 S） |
| 账号名 | 显示用户昵称或微信号 |
| 账号标签 | 标识当前是「知识助理」还是「WeKnora」 |
| 设置按钮 | 点击弹出设置面板 |
| 退出按钮 | 退出当前账号，返回登录选择页 |

### 3.3 设置面板

- 点击设置按钮后，以**底部弹出面板（Bottom Sheet）**形式展示
- 内容包含：
  - 用户头像与账号信息
  - 知识助理账号：显示微信昵称、微信号
  - WeKnora 账号：显示服务地址、API Key（可编辑修改）

### 3.4 知识库对话

主界面核心功能区，提供**聊天输入框**：

- 文本输入区域，支持多行输入
- 工具栏按钮：知识库选择、快速模式切换
- 发送按钮（有内容时激活为绿色）
- 发送后打开侧边栏对话面板，展示问答结果
- 支持快捷键发送（Enter 发送，Shift+Enter 换行）

### 3.5 网页内容采集

#### 3.5.1 保存当前页面

- 选择目标知识库
- 一键保存当前页面 URL 到知识库

#### 3.5.2 智能剪藏（全窗口截取）

- 自动提取当前页面的核心内容
- AI 智能识别正文、去除广告和导航等干扰信息
- 提取结果保存到知识库

#### 3.5.3 选择剪藏（部分窗口截取）

- 用户在页面上手动框选/圈选想要保存的区域
- 支持选中文字内容提取
- 选中内容保存到知识库

### 3.6 Markdown 快速记录

- 支持在插件内快速编写 Markdown 内容
- 编写完成后保存到指定知识库
- 作为快速笔记/速记工具使用

## 4. 交互流程

```
安装插件
  │
  ▼
首页：选择登录方式
  ├── 知识助理 → 自动获取微信信息 → 自动登录 → 主界面
  └── WeKnora → 填写服务地址 + API Key → 登录 → 主界面

主界面
  ├── 对话输入框 → 发送问题 → 侧边栏显示回答
  ├── 保存当前页面 → 选择知识库 → 保存
  ├── 智能剪藏 → 自动提取页面内容 → 保存
  ├── 选择剪藏 → 手动框选区域 → 保存
  ├── Markdown 速记 → 编写内容 → 保存到知识库
  ├── 设置 → 查看/修改账号信息
  └── 退出 → 返回登录选择页
```

## 5. 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘/Ctrl + Shift + K` | 打开侧边栏 |
| `⌘/Ctrl + Shift + A` | 快速提问 |

## 6. 技术栈

| 项目 | 技术选型 |
|------|----------|
| 扩展标准 | Chrome Extension Manifest V3 |
| 前端 | HTML + CSS + JS（外部文件，符合 CSP） |
| 状态存储 | chrome.storage.local |
| 消息通信 | chrome.runtime.sendMessage |
| 侧边栏 | chrome.sidePanel API |
| 内容脚本 | content.js + content.css |
